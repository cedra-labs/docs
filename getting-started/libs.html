Prerequisites: Rust and Node.js Installation

Ready to build on Cedra? Let's get your development environment set up! üöÄ

You'll need two powerful tools in your arsenal:

ü¶Ä Rust - The backbone for building Cedra from source and crafting Move modules
üì¶ Node.js - Your gateway to DApp development and frontend magic

üéØ Version Requirements‚Äã

For the smoothest Cedra experience, we recommend:

  • Rust: 1.86.0 or higher
  • Node.js: 18.x LTS or 20.x LTS
  • npm: 8.0.0 or higher

ü¶Ä Installing Rust‚Äã

Platform Support

Rust runs beautifully on Windows 10/11 (64-bit), macOS 10.12+, and Linux (kernel 3.2+)

  • üçé macOS
  • ü™ü Windows
  • üêß Linux

Via Homebrew Way. One command, and you're done!

brew install rust

The Official Installer Route Download and run the official installer

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Don't forget to activate Rust in your current shell:

source "$HOME/.cargo/env"

Official Installer (Recommended)‚Äã

  1. Grab rustup-init.exe
  2. Run it and follow the friendly prompts
  3. Select "1) Proceed with installation (default)"
  4. Give your terminal a fresh restart

Chocolatey Lovers. Sweet and simple‚Äã

choco install rust

Official Installer (Universal)

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Activate Rust in your shell:

source "$HOME/.cargo/env"

Via Package Manager

# Update and install in two steps
sudo apt update
sudo apt install rustc cargo

‚úÖ Verify Your Rust Installation‚Äã

Check if everything's working

rustc --version
cargo --version

üìö Need more details? Check out rust-lang.org/tools/install

üì¶ Installing Node.js‚Äã

Time to set up your JavaScript runtime! We recommend Node.js 18.x LTS or 20.x LTS for maximum stability.

  • üçé macOS
  • ü™ü Windows
  • üêß Linux

Homebrew (Lightning Fast)‚Äã

brew install node

Node Version Manager (For the Power Users)‚Äã

  1. Get nvm up and running:

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

  2. Refresh your shell:

    source ~/.bashrc

  3. Install the latest LTS:

    nvm install --lts
    nvm use --lts

Traditional Installer‚Äã

Prefer clicking? Download from nodejs.org and follow the wizard!

Official Installer

  1. Head to nodejs.org
  2. Download the LTS version
  3. Run the installer (keep "Add to PATH" checked!)

Chocolatey Magic

choco install nodejs

Version Manager (nvm-windows)

  1. Download nvm-windows
  2. Install it, then run as Administrator: 
    nvm install lts
    nvm use lts

NodeSource Repository (Recommended)‚Äã

# Add NodeSource repo and install
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

Node Version Manager (Flexible Option)‚Äã

  1. Install nvm:

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

  2. Reload your shell config:

    source ~/.bashrc

  3. Install Node.js:

    nvm install --lts
    nvm use --lts

üîß Final Touches‚Äã

  1. Supercharge npm:

    # Always use the latest npm
    npm install -g npm@latest

  2. Fix npm permissions (Linux/macOS):

    # Create a safe space for global packages
    mkdir ~/.npm-global
    npm config set prefix '~/.npm-global'
    echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
    source ~/.bashrc

  3. Install developer essentials:

    # The holy trinity of JavaScript tools
    npm install -g yarn pnpm typescript

‚úÖ Verify Node.js is Ready‚Äã

npm --version
