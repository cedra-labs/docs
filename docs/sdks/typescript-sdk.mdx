## Overview

The Cedra TypeScript SDK will provide a complete TypeScript/JavaScript interface for building applications on Cedra, including:

- Full TypeScript type safety
- Transaction building and submission
- Smart contract interactions
- Wallet integration
- Event listening and subscriptions
- DEX and DeFi protocol integrations
- React hooks for frontend development
